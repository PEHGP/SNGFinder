# Demo
The genome has been masked for repeats.  
The cactus alignment results in maf format have also been generated.  
Just run SNGFinder.  
```
SNGFinder --prefix saccer --maf YeastSp9_ref_Saccharomyces_cerevisiae.maf.gz --target target --refspecies Saccharomyces_cerevisiae
```
`saccer_NewGenesOrphan.xls`, `saccer_NewGenesParalog.xls` and `saccer_NewGenesRepeat.xls` are the results.
